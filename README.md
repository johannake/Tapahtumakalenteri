# 📅 Tapahtumakalenteri

Yksinkertainen web-pohjainen tapahtumakalenteri, jonka avulla voi lisätä, selata ja poistaa tapahtumia kuukausittain.

Rakennettu osana web-kehityksen opiskelua — tavoitteena oppia PHP:n ja MySQL:n yhteiskäyttöä

## Teknologiat

- **PHP** — palvelinpuolen logiikka ja tietokantakyselyt
- **MySQL** — tapahtumien tallennus
- **HTML & CSS** — rakenne ja ulkoasu
- **XAMPP** — paikallinen kehitysympäristö

## Ominaisuudet

- Tapahtumien lisäys, selaus ja poistaminen
- Kuukausinavigaatio
- Lomakkeen palvelinpuolen validointi
- SQL-injektiosuojaus (PDO prepared statements)
- XSS-suojaus (htmlspecialchars)
- Responsiivinen ulkoasu

## Opittua

Projektin aikana opin PDO:n käytön tietokantayhteyksissä sekä lomakkeiden käsittelyn PHP:llä.
